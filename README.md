# CryptoTerminal — Premium Trading Dashboard

A distinctive cryptocurrency trading dashboard with a bold "Cyber Terminal" aesthetic featuring neon accents, grid backgrounds, and professional trading interfaces.

## Framework

Static HTML with Vanilla JavaScript and Chart.js for candlestick visualization

## Preview

Configured to run on `0.0.0.0:8080` for k8s HTTPRoute access.

## Theme

**Cyber Terminal Dark** — Deep space blacks (#0a0e14) with electric neon green (#00ff9d) and cyan (#00f7ff) accents. Features:
- Grid pattern background with radial fade
- Layered card surfaces with subtle grain texture
- Glowing borders and shadows on interactive elements
- Sharp neon typography (JetBrains Mono + Orbitron)
- Depth through gradients and multi-layer shadows
- Live data pulse animations
- Asymmetric layout with premium fintech aesthetic

## Features

- **Left Sidebar**: Coin watchlist with live sparkline mini-charts
- **Main Chart**: Interactive candlestick chart with time period controls
- **Order Book**: Live bid/ask spread with depth visualization bars
- **Recent Trades**: Real-time trade feed with buy/sell indicators
- **Top Navigation**: Portfolio value and 24h performance tracking
- **Live Updates**: Simulated real-time data updates every 1-3 seconds
